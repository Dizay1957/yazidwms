# Screenshot Guide

This directory is reserved for real YazidWMS screenshots. Do not commit fake or empty PNG files.

Recommended screenshots:

- `login.png`
- `dashboard.png`
- `products.png`
- `inventory.png`
- `warehouses.png`
- `purchase-orders.png`
- `sales-orders.png`
- `reports.png`
- `users.png`
- `swagger.png`

Capture guidance:

- Use local demo data rather than personal or production data.
- Hide secrets, tokens, private email addresses, and local environment values.
- Use a consistent desktop browser size, preferably `1440x900` or similar.
- Capture the main content clearly, without browser developer tools open.
- Prefer light mode or one consistent theme across all screenshots.
- Make sure Arabic/RTL screenshots do not expose private data if included.
- Add `dashboard.png` first, then uncomment the dashboard image block in the root README.
