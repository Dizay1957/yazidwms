# Screenshot Guide

This directory stores real YazidWMS screenshots used in the repository documentation.

Included screenshots:

- `login.png`
- `dashboard.png`
- `products.png`
- `inventory.png`
- `warehouses.png`
- `purchase-orders.png`
- `sales-orders.png`
- `reports.png`
- `users.png`

Capture guidance:

- Use demo or local development data only.
- Hide secrets, tokens, and private contact details.
- Keep a consistent desktop browser size, preferably around `1440x900`.
- Capture the main content clearly without browser developer tools.
- Keep the interface readable and avoid clipped dialogs or menus.
